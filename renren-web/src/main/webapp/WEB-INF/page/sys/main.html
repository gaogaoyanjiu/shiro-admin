基本信息

   项目介绍

  • 一个轻量级的Java快速开发平台，能快速开发项目并交付
    •             **技术选型：**
            - 核心框架：Spring Framework 4.3
            - 安全框架：Apache Shiro 1.3
            - 视图框架：Spring MVC 4.3
            - 持久层框架：MyBatis 3.3
            - 定时器：Quartz 2.2
            - 数据库连接池：Druid 1.0
            - 日志管理：SLF4J 1.7、Log4j
            - 页面交互：Vue2.x
