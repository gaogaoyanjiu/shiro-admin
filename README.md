# shiro-admin
shiro后台权限管理系统(集成了权限控制、日志管理、文件上传、sql监控、代码生成器)
 **技术选型：**
            - 核心框架：Spring Framework 4.3
            - 安全框架：Apache Shiro 1.3
            - 视图框架：Spring MVC 4.3
            - 持久层框架：MyBatis 3.3
            - 定时器：Quartz 2.2
            - 数据库连接池：Druid 1.0
            - 日志管理：SLF4J 1.7、Log4j
            - 页面交互：Vue2.x
